# Full-Stack Monorepo

A unified monorepo containing both backend NestJS libraries and frontend/mobile client libraries.

## Structure

```
packages/
├── nestjs/                          # NestJS backend libraries
│   ├── logger/                      # @adatechnology/logger (Winston + AsyncLocalStorage)
│   ├── cache/                       # @adatechnology/cache (Redis/InMemory)
│   ├── http/                        # @adatechnology/http-client
│   ├── shared/                      # @adatechnology/shared (decorators, utilities)
│   ├── keycloak/                    # @adatechnology/auth-keycloak
│   ├── keycloak-admin/              # @adatechnology/keycloak-admin
│   └── example/                     # Example NestJS application
│
└── frontend/
    └── logger-client/               # @adatechnology/logger-client (isomorphic request tracing)
```

## Quick Start

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm run build:all

# Build only NestJS packages
pnpm run build:nestjs

# Build only frontend packages
pnpm run build:frontend

# Build and watch packages
pnpm run watch:packages

# Build example app
pnpm run build:example
```

## Key Concepts

### TraceStack (AsyncLocalStorage)
Hierarchical call tracing across services:
```
[requestId][timestamp][projectName:version][Class1.method][Class2.method][LEVEL] - message
```

### Distributed Tracing
- **Request ID**: Generated by frontend/mobile, propagated via headers (x-request-id)
- **Trace ID**: Generated by backend via OpenTelemetry, returned in response headers (x-trace-id)
- **Correlation**: Frontend stores trace ID for subsequent requests

### Frontend/Mobile Logger Client (@adatechnology/logger-client)
Isomorphic client for React Native, web, Node.js, and Electron:
- Wraps `fetch()` to add request correlation headers
- Stores trace ID from server responses
- Logs with [requestId][traceId] prefix for correlation with backend

## Services Using This Monorepo

- **domestic-backend-api**: Main API service
- **domestic-backend-bff**: Backend-for-frontend GraphQL gateway
- **domestic-backend-cron**: Scheduled jobs
- **domestic-backend-worker**: RabbitMQ consumer service
- **cawme-client**: Mobile app (uses @adatechnology/logger-client)
- **cawme-web**: Web app (uses @adatechnology/logger-client)
